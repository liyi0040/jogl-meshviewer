Operation Manual

1.Rotate the object:
Hold left mouse button to drag (not to much)
Release
Hold the window and shake several times till it repaint
(I know it's werid, sorry for I didn't find the auto-repaint function in java T_T)
Keep shaking so that it can rotate automaticlly with the magnitude and direction you dragged
Re-drag if you want to chage the rotation

2. Translate the object:
Hold middle mouse button to drag (if you want it go up, move your mouse bottom up, but not to much)
The other operations is similar to Rotation operation

3.Zoom in/out the object:
Hold right mouse button to drag
bottom-up to zoom out and top-down to zoom in
How much you will zoom depends on the distance your mouse move
